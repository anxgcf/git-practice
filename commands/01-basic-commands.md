# Git Basic Commands

This document explains the most commonly used Git commands
and when to use them.

---

## 1. git init
Initializes a new Git repository in the current folder.

```bash
git init
```

---

## 2. git status
Shows the current state of the working directory and staging area.

```bash
git status
```

---

## 3. git add
Stages changes for the next commit.

```bash
git add file.txt
git add .
```

---

## 4. git commit
Saves staged changes with a message.

```bash
git commit -m "feat: add initial notes file"
```

---

## 5. git log
Displays commit history.

```bash
git log
git log --oneline
```

---

## 6. git diff
Shows differences between files.

```bash
git diff
git diff --staged
```

---

## 7. git clone
Creates a local copy of a remote repository.

```bash
git clone git@github.com:username/repository.git
```

---

## 8. git pull
Fetches and merges changes from remote repository.

```bash
git pull origin main
```

---

## 9. git push
Uploads local commits to remote repository.

```bash
git push origin main
```
---

## 10. git rm
Removes a file from Git and the working directory.

```bash
git rm file.txt
```
